## Por qué este directorio existe

Las decisiones de arquitectura sin investigación son suposiciones.
Este directorio preserva el razonamiento detrás de cada decisión —
incluyendo los caminos descartados y por qué.

Un agente que lee `docs/adr/ADR-X.md` entiende la decisión.
Un agente que además lee la investigación correspondiente aquí
entiende **por qué no se eligieron las otras opciones**.
