# CORTEX — Knowledge Distillation Layer

CORTEX es la capa de persistencia del conocimiento de ZENTROPY.
Convierte conversaciones, decisiones y errores en nodos del grafo cognitivo.

## Estructura
- `ideas/` — Capturas brutas de sesiones de ideación
- `specs/` — Open Specs de proyectos/features
- `patterns/` — Patrones recurrentes identificados
- `errors/` — Post-mortems de errores importantes
- `decisions/` — Decisiones tomadas con contexto
- `references/` — Fuentes externas indexadas
- `projects/` — Estado de cada proyecto activo

## Cómo usar
MNENTROPY procesa y destila a Supabase automáticamente (12pm + 11pm).
Para captura manual urgente: `zentropy mnentropy run --now`
